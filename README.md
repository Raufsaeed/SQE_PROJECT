# Bridal Dress Shopping App

A simple app to buy bridal dresses with WhatsApp direct ordering.

## Features
- Login/Signup system
- 2 bridal dresses (Light Pink $99, White $89)
- Direct WhatsApp ordering

## How to use
1. Download APK from Releases
2. Install on Android phone
3. Click BUY NOW to order on WhatsApp
